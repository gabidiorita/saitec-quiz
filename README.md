# Saitec Quiz

Aplicación web libre para crear quizzes de hasta 15 preguntas y jugar con hasta 10 personas a la vez.

## Probar en este ordenador

1. Instala Node.js (versión 20 o superior).
2. Abre una terminal en esta carpeta y ejecuta `npm install`.
3. Ejecuta `npm start`.
4. Abre `http://localhost:3000` en el navegador.

Para jugar desde distintos lugares hay que publicar la aplicación en un servicio web compatible con Node.js (por ejemplo Render). El código no usa cuentas, anuncios ni servicios de pago.

## Publicarla online sin instalar nada en Saitec

La aplicación incluye `render.yaml`, por lo que está preparada para publicarse en Render. La persona que la publique necesita una cuenta de GitHub y una cuenta de Render; los jugadores solo necesitan el enlace y un navegador.
